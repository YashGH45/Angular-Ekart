About E-Kart

Welcome to E-Kart – your go-to demo for exploring E-commerce with Angular! This lightweight project is designed to showcase how modern web technologies can be used to build a seamless online shopping experience.

🚀 Key Features

  • Product browsing with clean UI
  • Detailed product pop-ups
  • Add to Cart functionality
  • Responsive design for all devices

🛠️ Tech Stack

  • Angular & TypeScript
  • CSS & Angular Material

🎯 Project Purpose

This project is a learning initiative to understand core Angular concepts including routing, component communication, API integration, and UI/UX best practices. It aims to simulate a real-world shopping experience in a beginner-friendly manner.
